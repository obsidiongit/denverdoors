# Our Work Specifications

## Content

### Overview
**Title:** Our Work
**Subtitle:** A Gallery of Recent Projects

### Project Categories
- **Tailored Interior Door Installs:** Before and afters of door replacements.
- **Custom Built-Ins:** Mudrooms, living rooms, and storage solutions.
- **Railing Systems:** Staircase updates and new railing installs.
- **New Build & Remodeling:** Large scale finish carpentry packages.
- **Specialty Accent Walls:** Shiplap, wainscoting, and feature walls.
- **Barn Doors:** Sliding barn door installations.

## Layout Reference
- **Gallery Grid:** Masonry or strict grid layout of project photos.
- **Filter Tabs:** Filter by category (All, Doors, Built-ins, Trim, etc.).
- **Lightbox:** Click image to expand view.

## Components to Use
- **Gallery:** `ProjectGallery` (Grid with filtering logic)
- **Image Card:** `PortfolioCard` (Image with hover overlay showing project type)
